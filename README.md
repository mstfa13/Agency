### Run Locally

To run **Apex Labs** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/mstfa13/apex-labs.git
```

Windows:

```bash
git clone https://github.com/mstfa13/apex-labs.git
```
